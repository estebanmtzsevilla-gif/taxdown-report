# TaxDown - Análisis Visual Interactivo

Análisis estratégico de crecimiento y experiencia de usuario para TaxDown México.

## Estructura del Proyecto

- `TaxDown_claude.html` - Archivo principal HTML con el reporte completo
- `styles.css` - Estilos CSS extraídos y organizados
- `index.html` - Archivo HTML adicional

## Características

- Visualización interactiva con scrollytelling
- Mapa de México con animaciones
- Panorama competitivo con visualizaciones de burbujas
- Secciones de análisis de mercado, fricción y valor
- Plan integral de crecimiento con acordeón interactivo
- Diseño responsive y moderno

## Tecnologías

- HTML5
- CSS3 (con variables CSS y animaciones)
- JavaScript (IntersectionObserver para animaciones al scroll)
- Font Awesome para iconos
- Google Fonts (Albert Sans)

## Uso

Simplemente abre `TaxDown_claude.html` en tu navegador para ver el reporte completo.
